# Приложения склад

## Функции
- Список товаров на складе
- Подробная информация о товаре

## Технологи
Для работы проект использует открытые библиотеки и стандарты современного програмирования и иконки с открытого доступа
- Retrofit
- LiveData
- NavСomponent
- Moshi
- Corountines
- Clean architecture
- Dagger 2
- MVVM
